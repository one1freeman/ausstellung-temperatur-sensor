##  Temperatursensor für eine Ausstellung des GMI

Im Rahmen der Ausstellung soll ein Modellhaus durch ein funktionales Modell einer Wärmepumpe aufgeheizt werden.
Dieser Mikrocontroller verbindet sich mit dem WLan Access Point des Haupt-Mikrocontrollers um alle 5 Sekunden per HTML Post-Request die gemessene Temperatur im Modellhaus zu schicken.
